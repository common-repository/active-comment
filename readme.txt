= Features of this Plugin =
* A clean, user friendly WordPress Interface/Layout for both user and admin.
* Users can comment to any post by simply login/register from same page/post.
* Like/Dislike Feature for end user in commenting system.
* Reply for any comment by a register user to any page.
* Single Sign-On when user Login at commenting interface as well as Login at WordPress website simultaneously.

== Installation ==
1. Login to WordPress Admin Section.
2. Navigate to "Add New" under "Plugins" Menu at left menu bar.
3. Find Plugin with "Active Commenting" name.
4. Install "Active Commenting" Plugin by hit install button.
5. Now hit Activate Button to active this plugin for your WordPress website.
